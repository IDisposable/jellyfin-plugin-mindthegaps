Mind the Gaps

Choose what the gap scanner looks for. Results appear on the Mind the Gaps ToDo List.

What to scan

Comma-separated studio names to complete, for example A24, Studio Ghibli. Resolved to TMDB at scan time.
Alternative to names: comma-separated TMDB company ids, for example 41077 (A24), 10342 (Studio Ghibli).
Comma-separated TMDB keyword ids to complete.

Where to watch

Powers the "Where to watch" links and the report's "Hide items with no sources" filter. Look-ups run on demand per item, or as a background pass from the report (the "Look up where to watch" button), never during the scan.

Limits

Caps how many similar titles each owned item contributes (recommended max 100).
Caps how many missing episodes are listed per show. 0 lists them all.
Filmography and recommendation scans rotate stalest-first within this cap, so coverage builds up over runs; raise this to cover a large cast and crew faster (each person is one cached TMDB lookup; recommended max 10000).
Forgets which items were scanned recently, so the next scan starts a fresh coverage cycle treating everything as never-scanned. Does not delete any gaps or dismissals. Stale entries for items that left the library are pruned automatically, so you rarely need this.

Experimental: virtual items

The plugin can write pathless "virtual" placeholder items into your library so missing entries render greyed-out, the way missing episodes do. It is a temporary stand-in for proper server support; the server will not reconcile or garbage-collect these for you. Mint individual gaps, or a multi-select batch, from the Mint buttons on the report page. Use Remove minted movies here to undo everything at once.
